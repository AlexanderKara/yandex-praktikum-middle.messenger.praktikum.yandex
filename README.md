###Команды для запуска:

Проект собирается в [Parcel](https://parceljs.org/):

```bash
npm run build
```

Сборка и запуск статического сервера на Express по адресу http://localhost:3000/:

```bash
npm run start
```

Сборка проекта в Parcel по адресу http://localhost:1234:

```bash
npm run dev
```
